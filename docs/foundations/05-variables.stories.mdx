# Theme Variables

Avaialable for use in React Components and Styled Components.

### App Constants

```jsx
import { appPrefix, libVersion } from '@qnt/ui';
```

### Spacing Scales

```jsx
import { theme } from '@qnt/ui';

const {
    Spacing1,
    Spacing2,
    Spacing3,
    Spacing4,
    Spacing5,

    Spacing9,
```
