# Typography

QNT UI library does not include any fonts of its own; it will use the default sans-serif operating system font.

The base font size for QNT UI web applications is 14px (0.875rem). This should be the default type size for most short strings of text which are not headings or titles.

## Font Defaults

| Family | Px | Rem | Weight |
| ------ | -- | --- | ------ |
| Sans-Serif | 14 | 0.875 | Regular |
